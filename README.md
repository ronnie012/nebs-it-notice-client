# Notice Management Client

## Overview
Frontend dashboard for managing notices with creation, listing, filtering, and publishing controls.

## Tech Stack
- React (Vite)
- TailwindCSS
- React Router

## Installation
npm install
npm run dev

## Environment Variables
Create `.env` file:
VITE_API_BASE_URL=https://nebs-it-notice-client.vercel.app/

## Deployment
Deploy on Vercel as a standalone frontend.
